STAN - subSTance flow ANalysis

What is STAN?

In this course we use the free STAN software. STAN (subSTance flow ANalysis) is used for modelling and visualizing material flow analyses by companies, government agencies and academic users.

STAN enables the modelling of systems which are composed of processes, flows and system boundaries. Different levels can be defined, such as goods, materials and energy. The flows can be displayed in such a way that their width is proportional to their value, which enables intuitive visual evaluation (e.g. which are the most important flows?).

Installation

STAN works only on Windows, but see work around below. You can get and use STAN as follows:

  • Version 1: Download the software from www.stan2web.net (you have to register on the website).

  • Version 2: If the registration and/or download do not work, you find the installation file on Moodle.

  • Version 3: You can run STAN also on the Citrix Workspace within the FHNW network. You found information about Citrix here (in German).

Anleitungen und Tutorials

  • The STAN manual is available on https://www.stan2web.net/downloads/documentation after registration on the STAN website.

  • You also find useful video tutorials here: https://www.stan2web.net/support/videos
